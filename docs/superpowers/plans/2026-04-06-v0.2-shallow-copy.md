# v0.2 — Shallow `with()` Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Generate a `with()` method on abstract base classes that creates a new frozen instance with selective property overrides.

**Architecture:** The emitter gains a `with(overrides: Partial<Params>): this` method that spreads current property values with overrides and constructs a new instance via `this.constructor`. No changes to the parser, generator, runtime, or CLI — only the emitter and its tests change. Behavior tests verify the end-to-end contract.

**Tech Stack:** TypeScript 6, ESM, Bun test runner, ts-morph

    const { Person } = await import('./fixtures/person.ts');
    const p1 = new Person({ firstName: 'John', lastName: 'Smith', age: 30 });
    const p2 = p1.with({ firstName: 'Jane' });
    expect(p2.firstName).toBe('Jane');
    expect(p2.lastName).toBe('Smith');
    expect(p2.age).toBe(30);
  });

  it('returns a different instance', async () => {
    const { Person } = await import('./fixtures/person.ts');
    const p1 = new Person({ firstName: 'John', lastName: 'Smith', age: 30 });
    const p2 = p1.with({ firstName: 'Jane' });
    expect(p2).not.toBe(p1);
  });

    const { Person } = await import('./fixtures/person.ts');
    const p1 = new Person({ firstName: 'John', lastName: 'Smith', age: 30 });

    expect(p1.firstName).toBe('John');

  it('returned instance is frozen', async () => {
    const { Person } = await import('./fixtures/person.ts');

    expect(Object.isFrozen(p2)).toBe(true);

    const { Person } = await import('./fixtures/person.ts');

- [ ] **Step 3: Add `emitWithMethod()` to the emitter**

In `src/generator/emitter.ts`, add a new function `emitWithMethod` and call it from `emitClassBody`.

Add after the existing `emitClassBody` function:

```ts
function emitWithMethod(cls: ParsedFreezedClass): string {
  const currentProps = cls.properties
    .map((p) => `      ${p.name}: this.${p.name},`)
    .join('\n');

  return `  with(overrides: Partial<${cls.className}Params>): this {
    const Ctor = this.constructor as new (params: ${cls.className}Params) => this;
    return new Ctor({
${currentProps}
      ...overrides,
    });
  }`;
}
```

Then modify `emitClassBody` to include the `with()` method. Replace the current `emitClassBody` function body with:

```ts
function emitClassBody(cls: ParsedFreezedClass): string {
  const readonlyFields = cls.properties
    .map((p) => `  readonly ${p.name}!: ${p.type};`)
    .join('\n');

  const assignments = cls.properties
    .map((p) => `    this.${p.name} = params.${p.name};`)
    .join('\n');

  const withMethod = emitWithMethod(cls);

  return `export abstract class ${cls.generatedClassName} {
${readonlyFields}

  constructor(params: ${cls.className}Params) {
${assignments}
    Object.freeze(this);
  }

${withMethod}
}`;
}
```
